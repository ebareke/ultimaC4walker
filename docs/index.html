🚶 ultimaC4walker

Ultimate walker for the four chromatin-profiling assays

ChIP-seq · CUT&RUN · CUT&Tag · ATAC-seq — FASTQ to peaks, tracks & QC

GitHub Usage Nextflow Containers Example

What it does

ultimaC4walker takes raw FASTQ to peaks, coverage tracks and a full QC report in one command, for four chromatin assays. A single portable Bash engine (c4walker) drives three run modes — standalone, container, and Nextflow — from the same code path, so they all execute identical commands.

Four assays

ChIP-seq, CUT&RUN, CUT&Tag, ATAC — each with the right MACS2 strategy.

FASTQ → results

FastQC → BWA → filter → dedup → MACS2 → bigWig → FRiP → MultiQC.

Local & HPC

local, slurm, pbs from one config.

Tool-free dry-run

--dry-run prints every command; CI-verified on a bare runner.

Quick start

Standalone

git clone https://github.com/ebareke/ultimaC4walker.git
cd ultimaC4walker
./bin/c4walker run -c config/config.yaml -s config/samples.tsv --dry-run

Container

docker build -t ultimac4walker:1.0.0 .
docker run --rm -v "$PWD":/data ultimac4walker:1.0.0 \
    run -c /data/config/config.container.yaml -s /data/config/samples.tsv

Nextflow

nextflow run nextflow/main.nf \
    --samplesheet samples.csv --fasta genome.fa -profile docker
